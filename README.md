# COLCOM: playbooks for a fleet of Raspberry Pi Computers
This repository is for automating Pi configuration and package installation.


## Supported Hardware Revisions
- Raspberry Pi 4B - 1GB
- Raspberry Pi 4B - 2GB
- Raspberry Pi 4B - 4GB
- Raspberry Pi 4B - 8GB
- Raspberry Pi 3B
- Raspberry Pi 3B+
